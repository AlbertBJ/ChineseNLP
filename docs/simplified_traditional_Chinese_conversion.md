# Chinese Simplified/Traditional Conversion

## Background

Chinese Simplified/Traditional conversion converts simplified Chinese characters into traditional Chinese characters, and vice versa.


## Example

Input:

```
苟利国家生死以,岂因祸福避趋之.
```

Output: 

```
苟利國家生死以,豈因禍福避趨之.
```
