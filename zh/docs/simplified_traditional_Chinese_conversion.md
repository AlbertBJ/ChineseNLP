# 中文繁简转化  (Simplified/Traditional Conversion)

## 背景

简体中文/繁体中文转换将简体中文字符转换为繁体中文字符，或反之。

## 示例

输入:
```
苟利国家生死以,岂因祸福避趋之.
```

输出: 

```
苟利國家生死以,豈因禍福避趨之.
```
